# **App Name**: CateringStock

## Core Features:

- Catálogo de Artículos: Navega por la lista de artículos disponibles con sus códigos, descripciones y precios asociados.
- Formulario de Solicitud de Alquiler: Rellena un formulario de pedido para el material que deseas alquilar.
- Precios Dinámicos: El formulario calculará automáticamente los precios, por día e incluyendo posibles daños (existirá un campo para especificar los días y los daños).
- Asistente de Finalización de Pedidos: Usa una herramienta para ayudar a elegir los artículos necesarios para un evento basándose en una breve descripción (ej. evento corporativo para 50 personas, boda al aire libre para 100 personas, fiesta de cumpleaños para 20).
- Verificación de Stock en Tiempo Real: Indica en tiempo real el número de piezas que el usuario quiere alquilar, indicando las piezas disponibles/totales al usuario a modo de información.
- Revisión del Pedido: Después de rellenar el formulario, los usuarios revisarán sus pedidos antes de enviarlos.
- Envío del Pedido: Envía la solicitud de alquiler para que sea gestionada por un administrador. No se adjuntará ninguna base de datos a la aplicación.

## Style Guidelines:

- Color primario: El verde oscuro (#00703C) del logo, que representa frescura y sostenibilidad.
- Color de fondo: Blanco suave (#F8F8FF), proporcionando un lienzo limpio y moderno para el contenido.
- Color de acento: El verde más claro (#90EE90) para resaltar los elementos interactivos y la información importante, ofreciendo claridad visual.
- Combinación de fuentes: 'Open Sans' para los encabezados y 'Roboto' para el texto del cuerpo, asegurando la legibilidad y una estética moderna.
- Iconos de línea con esquinas redondeadas, coloreados en el verde primario, para mantener un estilo consistente y una interfaz fácil de usar.
- Diseño limpio, basado en tarjetas, para facilitar la búsqueda y la gestión de los elementos del inventario.
- Transiciones sutiles y animaciones de carga para mejorar la experiencia del usuario.
